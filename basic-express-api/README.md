# Basic Express API

This project is a simple API built using Node.js and the Express framework. It defines a single endpoint that responds with a greeting message.

## Project Structure

```
basic-express-api
├── src
│   └── app.js        # Entry point of the application
├── package.json      # npm configuration file
└── README.md         # Project documentation
```

## Installation

To get started, clone the repository and install the dependencies:

```bash
npm install
```

## Running the API

To start the server, run the following command:

```bash
node src/app.js
```

The server will log the port it's running on to the console.

## Endpoint

- **GET /**: Responds with "Hola mundo desde mi API".
