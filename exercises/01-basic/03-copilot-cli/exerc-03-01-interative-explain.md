# Ejercicio: Establecer una sesión de consulta interactiva con GitHub Copilot desde la línea de comandos (CLI)

## Objetivo

Aprender a utilizar el modo interactivo de Git

## Requisitos previos

- Cuenta de GitHub con acceso a Copilot de pago
- GitHub Copilot CLI instalado y configurado
- Política de organización habilitada si se usa a través de una organización
- Conocimiento básico de la línea de comandos
- Terminal o línea de comandos

## Estimación de tiempo

Tiempo estimado en complentarse: 5-10 minutos

## Instrucciones

### Caso de uso 1

Averiguar qué archivos y carpetas existen en el directorio de trabajo actual utilizando el modo interactivo de GitHub Copilot CLI

#### Paso 1: Arrancar la terminal

Abrir la terminal con la que se va a trabajar

#### Paso 2: Localizar el directorio de trabajo

Ubicarse en el directorio de trabajo donde se quiere utilizar GitHub Copilot CLI

#### Paso 3: Arrancar el modo interactivo

Ejecutar el siguiente comando:

```bash
copilot
```

Verificar que se ha "entrado" en el modo interactivo

En caso de que sea la primera vez que se utiliza, se pedirá iniciar sesión en GitHub y autorizar el acceso a Copilot CLI

#### Paso 4: Confirmar el mecanismo de confianza

Elegir la primera opción

Verificar que se ha entrado en el modo interactivo correctamente

#### Paso 5: Realizar una consulta

Escribir la siguiente consulta:

```
Explícame qué existe en este directorio
```

Se mostrará la explicación del contenido del directorio actual

#### Paso 6: Salir del modo interactivo

Escribir el siguiente comando slash para salir del modo interactivo:

```
/exit
```

Verificar que se muestran las estadísticas de uso
