# Enfoque de aproximación de custom agents

Este ejercicios ayudará a descubrir los roles de "persona" que más internvienen en el día a día de un trabajador de tecnología

## Objetivo

Crear uno o varios custom agents que adapten el comportamiento de Copilot a tu estilo, tu rol y el tipo de proyecto

Configurar y practicar con los diferentes roles (por ejemplo: “explicador”, “revisor”, “mentor técnico”, etc.)

## Tareas

* Introducción conceptual (5 min)
  * Qué es un modo de chat y cómo amplifica el contexto de Copilot
  * Ejemplos de modos preconfigurados (debugger, code reviewer, trainer)
* Diseño colaborativo (20 min)
  * Definir los modos más útiles (10 min): mentor técnico, revisor de código, depurador, arquitecto, etc.
  * Crear un modo de chat para revisión de código
    * Definir su prompt base (tono, objetivo, formato de salida)
    * Ejercicio práctico: “Diseña un modo ‘Code Reviewer’ que dé feedback con foco en seguridad y buenas prácticas.”
* Simulación práctica (25 min)
  * Activar el modo en una sesión de chat y probarlo con fragmentos reales
  * Comparar las diferencias de respuesta con el modo estándar
* Optimización y documentación (10 min)
  * Ajustar el modo según los resultados : tono, estructura y precisión
  * Guardar la plantilla como “Modo personal de revisión”
